Regression Analysis

A class of machine learning problems where the objective is to predict a continuous or numerical outcome from the input data. It creates a link between input characteristics and a target variable.

+

Classification Analysis

The goal of classification problems is to assing labels to incoming data. Accurate predictions for new instances are made possible by the mapping of features to preset categories.

+

Description Analysis

A problem which involves providing in-depth knowledge or insights on input data. The objective is to recognize and present any patterns or trends in the data.

+

Clustering Analysis

This is the process of combining similar data points by their attributes in order to identify underlying structures or patterns.
